About SQLcaster

SQLcaster's "frontend" is a single HTML page with some hand-rolled vanilla JS and its "backend" is an autogenerated REST API accepting SQL queries via ROAPI hosted on Replit. Contributions and ideas would be awesome - here's the repo. Created by @whatrocks.
