About SQLcaster

SQLcaster's "frontend" is a single HTML page with some hand-rolled vanilla JS and its "backend" is an autogenerated REST API accepting SQL queries via ROAPI hosted on Replit. This site is intended to be fun, experimental way to explore messages ("casts") on the Farcaster protocol. Contributions and ideas would be wonderful - here's the open source repo. SQLcaster was created by @whatrocks.
